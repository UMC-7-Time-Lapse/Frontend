# 6th UMC HACKATHON 7조 Frontend

- **Commit type**
    
    ```
    <type> : <subject>
    
    <body>
    ```
    
    예시)
    
    ```
    [Feat]회원 가입 기능 구현
    
    SMS, 이메일 중복확인 API 개발
    ```
    
    - **type**
        - **`태그 : 제목`의 형태이며, `:`뒤에만 space가 있음**
            
            
            | type | description |
            | --- | --- |
            | Feat | 새로운 기능 추가 |
            | Fix  | 버그 수정 |
            | Docs  | 문서 추가/수정 |
            | Refactor  | 코드 리펙토링 |
            | Test  | 테스트 코드, 리펙토링 테스트 코드 추가 |
            | Move | 파일, 폴더 경로 변경 |
